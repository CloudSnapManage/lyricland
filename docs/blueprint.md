# **App Name**: Lyric Land

## Core Features:

- Lyrics Search: Allows users to search for lyrics by artist and track name.
- API Integration: Fetches lyrics from LRCLIB API and falls back to another public lyrics API if needed.
- Lyrics Display: Displays fetched lyrics in a formatted, readable block on the frontend.
- Error Handling: Displays a user-friendly message when lyrics are not found.

## Style Guidelines:

- Primary color: A vibrant indigo (#4F46E5) to evoke a sense of creativity and discovery.
- Background color: Light grey (#F9FAFB), offering a clean and unobtrusive backdrop that makes content easy to read
- Accent color: A calming teal (#38b2ac), drawing attention and reinforcing a sense of trust
- Headline font: 'Space Grotesk' sans-serif for a techy, modern feel. Body font: 'Inter' sans-serif for readability.
- Centered layout for key content, ensuring focus and optimal readability across devices.
- Subtle fade-in animations when displaying search results for a smooth user experience.
